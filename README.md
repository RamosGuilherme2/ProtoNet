Java Backend Learning Platform

Description
A learning platform for Java enthusiasts to practice backend development with APIs, database integrations, and other essential skills. This project focuses on building a solid backend foundation using Spring Boot, PostgreSQL, and hexagonal architecture principles.

Features
Java backend using Spring Boot
PostgreSQL database integration
Hexagonal architecture structure for packages and classes
RESTful APIs for CRUD operations
Modular and maintainable code suitable for learning and experimentation
Designed for learning and prototyping backend concepts
